# Bash Scripts

A collection of useful bash scripts and utilities.

## Overview

This repository contains various bash scripts for automation, system administration, and general productivity tasks.

## Getting Started

### Prerequisites

- Bash shell (version 4.0 or higher recommended)
- Unix-like operating system (Linux, macOS, WSL, etc.)

### Installation

Clone this repository:

```bash
git clone https://github.com/Berg_W2/bash.git
cd bash
```

Make scripts executable:

```bash
chmod +x *.sh
```

## Usage

add functions to bashrc:

```bash
./setup.sh
```

## Deletion

```bash
./delete.sh
```
